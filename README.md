# AoC-Anno-Domini-Toppled
Allows for scenario-specific flavor years.

# Features
```
{
    "yearOffset" : "0",
    "negativeYearNotation" : "BC",
    "positiveYearNotation" : ""
}
```
Edit config.json however you'd like to change default year appearance. To make scenario-specific year presets make a folder named exactly like the scenario with the edited config inside. Examples are included.

## Requirements
- BepInEx 5 x64 (Mono). Download from the official BepInEx releases. https://github.com/bepinex/bepinex/releases

## Install

1. Extract the BepInEx zip into the game folder (next to the exe). Run the game once.
     1a. To find the game's folder, right-click the game on Steam, go to Manage, and then         Browse Local Files.
2. Extract the mod’s zip into the plugins location so that:
BepInEx\plugins\Anno Domini Toppled - BC Date Mod exists.
3. Run the game. Check BepInEx/LogOutput.log for "Anno Domini Toppled loaded".
